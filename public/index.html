SAI — Prueba de Endpoints
Autenticación Sin sesión
Token: —
Listar Documentos — GET /api/documents
Consultar por ID — GET /api/documents/:id
Subir Documento — POST /api/documents/upload

Requiere token (iniciar sesión primero)

Si no se indica vigencia, se extrae del PDF

Editar Documento — PUT /api/documents/:id

Requiere token — Solo los campos que llenes serán modificados

Se suma a la fecha de vencimiento actual

Nueva Versión — POST /api/documents/:id/version

Requiere token — Sube un nuevo PDF como versión actualizada del documento

Si no se indica, se extrae del PDF

Eliminar Documento — DELETE /api/documents/:id

Requiere token — El documento se marcará como inactivo (no se borra físicamente)

Comparar Versiones — GET /api/documents/:id/compare

Requiere token (iniciar sesión primero). Compara el texto extraído de dos revisiones del mismo documento, mostrando cambios detallados estilo diff
